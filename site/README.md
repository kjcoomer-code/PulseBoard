# Public Site

This folder contains the public-facing pages for Oura app registration.

## Files

- `index.html`: product landing page
- `privacy.html`: privacy policy
- `terms.html`: terms of service
- `site.css`: shared styling

## Quick publish options

### GitHub Pages

1. Push the `site` folder contents to a repository.
2. In GitHub, enable Pages for the branch you publish from.
3. If publishing from the repository root, use these URLs:
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/`
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/privacy.html`
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/terms.html`

### Netlify

1. Drag the `site` folder into Netlify Drop or connect the repo.
2. Use the generated URLs for:
   - `/`
   - `/privacy.html`
   - `/terms.html`

## Suggested Oura app values

- Website: landing page URL
- Privacy Policy: privacy page URL
- Terms of Service: terms page URL
